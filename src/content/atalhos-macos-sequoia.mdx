# macOS Sequoia
#### Atalhos de Teclado

## Atalhos Básicos do Sistema

**⌘ + Space** - Abrir Spotlight (pesquisa)
**⌘ + Tab** - Alternar entre aplicativos abertos
**⌘ + `** - Alternar entre janelas do mesmo aplicativo
**⌘ + Q** - Fechar aplicativo
**⌘ + W** - Fechar janela atual
**⌘ + M** - Minimizar janela
**⌘ + H** - Ocultar aplicativo atual
**⌘ + Option + H** - Ocultar outros aplicativos

## Finder

**⌘ + N** - Nova janela do Finder
**⌘ + Shift + N** - Nova pasta
**⌘ + Delete** - Mover para Lixeira
**⌘ + Shift + Delete** - Esvaziar Lixeira
**⌘ + D** - Duplicar item selecionado
**⌘ + I** - Obter informações do item
**⌘ + 1/2/3/4** - Alternar visualizações (ícones/lista/colunas/galeria)
**⌘ + Up Arrow** - Ir para pasta pai
**⌘ + Down Arrow** - Abrir pasta/arquivo selecionado

## Edição de Texto

**⌘ + A** - Selecionar tudo
**⌘ + C** - Copiar
**⌘ + X** - Recortar
**⌘ + V** - Colar
**⌘ + Z** - Desfazer
**⌘ + Shift + Z** - Refazer
**⌘ + F** - Localizar
**⌘ + G** - Localizar próximo
**⌘ + Shift + G** - Localizar anterior

## Captura de Tela

**⌘ + Shift + 3** - Capturar tela inteira
**⌘ + Shift + 4** - Capturar área selecionada
**⌘ + Shift + 4 + Space** - Capturar janela específica
**⌘ + Shift + 5** - Abrir utilitário de captura de tela

## Dock e Aplicativos

**⌘ + Option + D** - Mostrar/ocultar Dock
**F11** - Mostrar Desktop
**⌘ + Mission Control** - Abrir Mission Control
**Control + Up Arrow** - Mission Control
**Control + Down Arrow** - App Exposé
**Control + Left/Right Arrow** - Alternar entre Desktops

## Safari

**⌘ + T** - Nova aba
**⌘ + Shift + T** - Reabrir aba fechada
**⌘ + W** - Fechar aba
**⌘ + R** - Recarregar página
**⌘ + L** - Focar na barra de endereços
**⌘ + D** - Adicionar aos favoritos
**⌘ + Shift + R** - Recarregar sem cache
**⌘ + [** - Voltar página
**⌘ + ]** - Avançar página

## Gerenciamento de Janelas

**⌘ + Control + F** - Tela cheia
**⌘ + M** - Minimizar janela
**⌘ + Option + M** - Minimizar todas as janelas
**F3** - Mission Control
**F4** - Launchpad

## Acessibilidade

**⌘ + Option + 8** - Ativar/desativar zoom
**⌘ + Option + +/-** - Aumentar/diminuir zoom
**⌘ + F5** - Ativar VoiceOver
**Control + Option + F8** - Abrir Utilitário VoiceOver

## Atalhos de Força

**⌘ + Option + Esc** - Forçar saída de aplicativo
**Control + ⌘ + Power** - Reiniciar forçado
**Control + ⌘ + Media Eject** - Sair de todos os apps e reiniciar

## Novidades do Sequoia

**⌘ + Shift + A** - Abrir pasta Aplicativos no Finder
**Control + ⌘ + Q** - Bloquear tela
**⌘ + Shift + .** - Mostrar/ocultar arquivos ocultos no Finder

## Dicas Úteis

- Segure **Shift** durante inicialização para Modo Seguro
- **⌘ + R** durante inicialização para Modo de Recuperação
- **Option** + clique no ícone WiFi para informações detalhadas da rede
- **Option** + clique no ícone da bateria para informações detalhadas da bateria
- **⌘ + Option + Power** para dormir imediatamente​​​​​​​​​​​​​​​​
